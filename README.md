# Aegis Agentic Workspace: Enterprise Multi-Agent Orchestrator

Aegis is a sophisticated enterprise-grade multi-agent system powered by LangGraph. It automates compliance audits, policy generation, and task management by orchestrating specialized AI agents to process company documents and data.

🚀 Key Features

Multi-Agent Pipeline: Sequential processing through Librarian (Docs), Compliance Officer (Policy), Synthesis Planner (LLM), and Task Ledger (SQL).

Vector Intelligence: Utilizes ChromaDB for semantic search across enterprise handbooks and security policies.

Human-in-the-Loop: A real-time SQLite backend allows managers to track, toggle, and resolve actionable tasks generated by the agents.

Responsive Control Center: A modern, Tailwind-powered UI dashboard that offers both an Interactive Simulation Mode and Live API Integration.

🛠 Tech Stack

Orchestration: LangGraph (State Machine Workflow)

LLM Engine: Google Gemini

Vector Store: ChromaDB

Database: SQLite (via SQLAlchemy)

Backend: FastAPI

Frontend: Tailwind CSS, Vanilla JS, FontAwesome

🏢 How an Enterprise Can Use This

Companies can drop their own internal policy documents, HR handbooks, and security guidelines into the data/ folder and re-run the seed script to instantly transform their static PDFs into an active, compliant agentic workflow.

Setup Instructions

Clone the repository.

Install dependencies: pip install -r requirements.txt

Configure API: Get your Gemini API Key from Google AI Studio and update your .env file.

Seed Database: Run python backend/scripts/seed_db.py to embed your documents and initialize the task ledger.

Launch Server: Run PYTHONPATH=backend uvicorn app.main:app --reload.

Interface: Open frontend/index.html in any browser to start managing enterprise tasks.
